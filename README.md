# AI Resume Analyzer

This project is an AI-based Resume Analyzer built using Python and Flask.

## Features
- Resume upload (PDF)
- Skill extraction
- Job role recommendation
- Skill gap analysis
- Mobile accessible web app

## Technologies Used
- Python
- Flask
- HTML/CSS
- AI Logic (Keyword Matching)

## How to Run
1. Install dependencies:
   pip install flask PyPDF2

2. Run the application:
   python app.py

3. Open browser:
   http://127.0.0.1:5000

## Future Scope
- AWS S3 cloud integration
- Machine learning based resume ranking
